- 浏览器支持原生的 ES + JS 工具使用编译型语言

### 旧方案的问题 webpack 等 bundle

- 开发服务器慢
- 采用先抓取——>再构建
- 构建内容存内存，重新构建再重载页面

### 怎么变快的

遇到不变的纯 JS：使用 esbuild
JSX CSS Vue：ESM

Vite 中 HMR 在原生 ESM 上执行
利用 HTTP 头

- 源码使用 304 协商缓存
- 依赖使用强缓存
